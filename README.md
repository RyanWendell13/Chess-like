## Technology
### Frontend
- React

### Backend
- Node JS
- Express
- MongoDB

## Technical Information:

### How To Run (Locally)
Create a .env file

    PORT = (Port number to run on)
    MONGO_URI = (MongoDB connection string)

Run the command: npm run initialize. Then run npm start.
